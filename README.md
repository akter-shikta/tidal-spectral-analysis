# Tidal Spectral Analysis

## Institution
Università degli Studi di Genova

## Author
Fahmida Akter Shikta

## Overview
This project investigates tidal constituents using Fourier spectral analysis on free surface elevation data from Punta Salute (Canale della Giudecca), Venice.

## Key Topics
- Fourier Transform (FFT)
- Spectral analysis
- Tidal harmonic reconstruction
- M2 and S2 tidal constituents
- Periodogram analysis

## Skills Demonstrated
- Coastal hydrodynamics
- Time-series analysis
- Signal processing
- Tidal constituent identification

## Key Findings
The analysis identified dominant semidiurnal tidal constituents and demonstrated accurate tidal signal reconstruction using harmonic components.
